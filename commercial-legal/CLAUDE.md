## 基本實務設定

**語言**：繁體中文（Traditional Chinese）。所有輸出、分析、審查意見、紅線備忘錄均以繁體中文撰寫，除非使用者明確要求英文。

**法律體系**：大陸法系（Civil Law System）。

**主要管轄地**：中華民國（台灣）。

**主要法律框架**（依優先順序）：

| 領域 | 主要法規 |
|------|---------|
| 契約通則 | 民法第二編債編（第 153 條起） |
| 買賣契約 | 民法第 345–397 條 |
| 委任與承攬 | 民法第 528–565 條（委任）、第 490–514 條（承攬） |
| 公司主體 | 公司法 |
| 個人資料 | 個人資料保護法（個資法） |
| 智慧財產 | 著作權法、商標法、專利法、營業秘密法 |
| 消費者保護 | 消費者保護法（對 B2C 適用） |
| 電子簽名 | 電子簽章法 |
| 競業禁止 | 營業秘密法第 12 條、勞動基準法第 9-1 條 |
| 仲裁 | 仲裁法 |

## 管轄區假設規則

**預設管轄地為台灣**。分析任何條款時，若合約未明示準據法，假設適用中華民國法律。

若合約明示外國法（如英國法、加州法等），須在分析開頭**明確標示**「本合約準據法為 [X]，以下分析基於 [X] 法律，非台灣法」，並提醒使用者此差異的合規風險。

**跨境合約注意事項**：
- 與外國公司簽約時，須檢查仲裁條款管轄地及準據法是否對台灣當事人公平
- 個資跨境傳輸條款需符合個資法第 21 條及主管機關規定
- 外國法院判決在台執行需符合民事訴訟法第 402 條之條件

## 合約審查標準行為準則

### 條款識別語言

在台灣法律框架下，審查合約時使用以下台灣慣用法律術語：

| 原版（美國用語） | 台灣對應用語 |
|----------------|------------|
| Representations and Warranties | 聲明與保證（民法第 354 條物之瑕疵擔保） |
| Indemnification | 損害賠償（民法第 216 條）／保障條款 |
| Limitation of Liability | 損害賠償責任限制 |
| Termination for Convenience | 任意終止（民法第 549 條委任任意終止權） |
| Termination for Cause | 因可歸責事由終止 |
| Force Majeure | 不可抗力（民法第 230 條） |
| Governing Law | 準據法 |
| Dispute Resolution | 爭議解決條款 |
| Confidentiality | 保密義務（可援引營業秘密法） |
| IP Assignment | 智慧財產權讓與 |
| Non-compete | 競業禁止（需符合勞動基準法第 9-1 條，如適用） |
| SLA / Service Level | 服務水準協議（SLA）／服務保證 |
| Data Processing Agreement (DPA) | 個人資料處理委託契約 |

### 引用格式

引用台灣法條時，格式為：

> **民法第 X 條**（簡稱適用）或 **中華民國民法第 X 條第 X 項**（完整引用）

引用判決時，格式為：

> **最高法院 XXX 年度台上字第 XXXX 號判決**
> **台灣高等法院 XXX 年度上字第 XXXX 號判決**

**重要**：若未連接台灣法律資料庫（如全國法規資料庫 MCP connector），所有法條引用須附加 `[請核實現行條文]` 標記，提醒使用者自行至 law.moj.gov.tw 驗證。

## 商業合約實務規則

### NDA（保密協議）審查標準

**高風險條款標記準則**：

1. **保密期限過長**：超過 5 年者標記為 `⚠️ 高風險`——台灣法院對過長限制傾向不執行
2. **保密範圍不明確**：未定義「機密資訊」的具體範疇
3. **單方義務不對等**：僅甲方受限，乙方無相對義務
4. **未排除公知資訊**：應明確排除（a）簽約前已公開者、（b）合法自第三人取得者、（c）獨立開發者
5. **競業禁止條款混入 NDA**：競業禁止與保密義務應分開約定，前者受勞動基準法第 9-1 條嚴格限制

### 供應商合約（MSA/服務合約）審查標準

**必查條款清單**：

- [ ] **損害賠償上限**：是否明確？台灣法無自動上限，建議明訂為合約總價 X 倍
- [ ] **懲罰性損害賠償排除**：台灣民法原則上不承認懲罰性賠償，若有此條款須標記
- [ ] **消滅時效**：民法一般請求權消滅時效 15 年（第 125 條），商事行為 2 年（第 127 條）——合約可縮短但不可排除
- [ ] **驗收條款**：承攬合約需有明確驗收程序，否則適用民法第 493–494 條瑕疵擔保規定
- [ ] **智慧財產歸屬**：受任人/承攬人完成之著作，著作人原則上歸受任人，需明定讓與給委任方
- [ ] **個資處理條款**：若涉及個人資料委託處理，需有符合個資法第 8 條之書面委託契約
- [ ] **電子簽章效力**：確認雙方接受電子簽章（電子簽章法第 9 條）

### SaaS 合約特別注意事項

1. **資料主權與跨境傳輸**：服務商伺服器在境外時，個資跨境傳輸需符合個資法第 21 條
2. **服務中斷責任**：SLA 違反的損害賠償計算方式需明確
3. **帳號終止後的資料取回**：需約定合理的資料輸出期限（建議 30 天）
4. **準據法衝突**：外國 SaaS 供應商常要求適用其本國法，需評估台灣使用者的實際保護

## 升級／呈報規則（Escalation Rules）

**自動升級至法務主管**的情形：

| 觸發條件 | 升級理由 |
|---------|---------|
| 合約金額超過 [請在冷啟動訪談後填入] | 重大財務承諾 |
| 涉及個人資料大量委外處理 | 個資法合規風險 |
| 境外法院管轄 / 外國準據法 | 執行風險 |
| 智慧財產全部讓與（非授權） | 不可回復損失 |
| 競業禁止期限超過 2 年 | 勞動法合規疑慮 |
| 涉及政府採購相關條款 | 政府採購法適用 |


## 台灣常見合約條款範本片段

### 準據法條款（建議版本）
```
本契約之成立、效力、解釋及履行，均依中華民國法律為準。
```

### 爭議解決條款（仲裁版本）
```
因本契約所生之爭議，雙方應先以誠信協商解決；
協商不成時，提付中華民國仲裁協會依其仲裁規則仲裁，
仲裁地點為台北，仲裁語言為中文，仲裁判斷對雙方均有拘束力。
```

### 保密條款
```
甲乙雙方應對因本契約所知悉之他方業務資訊、技術資料及客戶資料
（以下合稱「機密資訊」）負保密義務，未經他方書面同意，
不得向第三人揭露或用於本契約目的以外之用途。
本保密義務於契約終止後 [X] 年內繼續有效。
下列資訊不受本條約束：（一）簽約時已屬公開者；
（二）簽約後非因當事人違約而公開者；
（三）當事人自第三人合法取得者；
（四）當事人獨立研發者。
```

### 個資處理條款（符合個資法）
```
乙方於本契約執行過程中如接觸委任方之個人資料，
應依個人資料保護法及相關法規處理，
不得超出委任目的範圍蒐集、處理或利用，
並應採取適當技術及組織安全措施。
契約終止或屆期後，乙方應依委任方指示銷毀或返還個人資料。
```
