# /spotdraft-post-review

Pushes a review summary to SpotDraft as a contract comment тАФ OAuth-attributed to the reviewing attorney.

## Instructions

1. **Confirm write-enabled SpotDraft.** The default read-only connector returns `403` / `WRITE_DISABLED` on `add_contract_comment`. If write is disabled, surface:

   > SpotDraft write tools are not enabled on the connected server. To post review comments, use the write-enabled SpotDraft connector: in Claude Cowork, Settings тЖТ Connectors тЖТ remove SpotDraft тЖТ add SpotDraft (Write) using the URL `https://mcp.us.spotdraft.com/mcp?enable_write=true`. You'll re-authorize via OAuth. The write-enabled URL is documented in the commercial-legal README.

2. **Find the contract.** Call `get_contract_list` by name or counterparty from user input or the review context.

3. **Build the comment** from the review memo (triage level, playbook side, key findings). Use the comment format below.

4. **Irreversibility gate (hard тАФ cannot be skipped).** Display:

   ```
   ## Ready to post to SpotDraft

   Contract: [name] ([SpotDraft ID])
   Action: Add comment тАФ will be visible to all users with access to this contract
   Posting as: [authorized OAuth user]
   Content preview:
   ---
   [First 200 characters of the comment]
   ---

   тЪая╕П This action cannot be undone.

   Type YES to post, or anything else to cancel.
   ```

5. **Proceed only if the user types exactly `YES`.** No override, no `--force` flag.

6. **Call `add_contract_comment`** once confirmed.

## Purpose

After `vendor-agreement-review`, `nda-review`, or `saas-msa-review`, push a summary of findings as a comment on the SpotDraft contract record for audit trail and team visibility.

## Auth

OAuth тАФ the comment is posted as the authorized user (the lawyer who ran the review).

## MCP tools

| Tool | Mode | What it does |
|---|---|---|
| `get_contract_list` | Read | Find contract by name or counterparty |
| `add_contract_comment` | **Write** | Post review summary as comment |

## Write connector

Requires URL `https://mcp.us.spotdraft.com/mcp?enable_write=true` (not the default read-only entry in `.mcp.json`).

```
[Commercial Legal Review тАФ Claude] YYYY-MM-DD

Reviewed by: [authorized OAuth user]

```

## Irreversibility gate

Before `add_contract_comment`, the skill MUST show the confirmation prompt and wait for exact `YES`. Any other response cancels.
